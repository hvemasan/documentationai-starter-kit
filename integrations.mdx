## Bring Every Feedback Source into ProductBridge

ProductBridge integrates with the tools where your customers and teammates already talk about your product. Feedback captured through any integration flows into the same boards, deduplication, and AI analysis as feedback submitted directly — so nothing gets siloed.

To browse and connect integrations, click **Connect Sources** in the dashboard sidebar. The Integrations catalog lists every available integration grouped by category, with a **Connected only** filter. Click any card to open that integration's configuration page.

## How Integrations Work

Most integrations follow the same lifecycle:

1. **Connect** — Open **Connect Sources**, pick the integration, and authorize it (OAuth) or provide the required details (for example, a domain and API key, or a public listing URL).

2. **Capture** — Feedback flows in automatically via webhooks, scheduled syncs, or explicit submissions like slash commands and sidebar apps.

3. **Analyze** — Imported items run through ProductBridge's AI pipeline for detection, extraction, deduplication, and routing to the right board.

4. **Act** — Link posts to issues in your tracker, notify voters when statuses change, and push events to other tools through automations.

<Callout kind="tip">
  Start with the channel where most of your customer conversations happen — usually your support desk or Slack — then add review platforms and an issue tracker once feedback is flowing.
</Callout>

For custom integrations, ProductBridge also provides a REST API and outbound webhooks. See the [API Reference](/api-reference/introduction) to get started.
