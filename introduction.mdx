## Overview

ProductBridge simplifies how you create, organize, and manage project documentation. You get a centralized space to build comprehensive docs, collaborate with teams, and keep everything up-to-date without the hassle of scattered files or complex tools.

Whether you document APIs, user guides, or internal wikis, ProductBridge handles versioning, search, and sharing seamlessly. Start building your knowledge base today.

## Quick Start Guide

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.productbridge.com` with your email.
  </Step>
  <Step title="Set Up Workspace" icon="settings">
    Choose a template or start blank. Add your first document.
  </Step>
  <Step title="Invite Collaborators" icon="mail">
    Share your workspace link and manage permissions.
  </Step>
  <Step title="Publish Docs" icon="globe">
    Make your docs public or share privately.
  </Step>
</Steps>

Here is a simple integration example using the ProductBridge API:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.productbridge.com/v1/docs', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      title: 'My First Doc',
      content: 'Welcome to ProductBridge!'
    })
  });
  const doc = await response.json();
  console.log(doc.id);
  ```
  ```python
  import requests

  response = requests.post(
      'https://api.productbridge.com/v1/docs',
      headers={
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json'
      },
      json={
          'title': 'My First Doc',
          'content': 'Welcome to ProductBridge!'
      }
  )
  doc = response.json()
  print(doc['id'])
  ```
</CodeGroup>
