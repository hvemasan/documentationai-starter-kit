## Capture Feedback from Every Conversation

Your team communicates with users across multiple channels every day — support tickets, chat messages, community servers. ProductBridge integrates with popular communication platforms to automatically extract and import feedback, so nothing gets lost.

## How Imported Feedback Works

All feedback imported from communication channels is:

- **Tagged with its source** — You always know where feedback came from (e.g., "via Intercom", "via Slack")
- **Linked to a user** — If the message has a user email or identity, ProductBridge matches it to an existing user profile or creates a new one
- **Deduplicated** — If the same user submits similar feedback across multiple channels, ProductBridge groups them together
- **Available in Intelligence** — Imported feedback flows into [Feedback Intelligence](/core-concepts/feedback-intelligence) for automatic categorization and analysis

<Callout kind="info">
  Communication channel integrations are one-way by default — feedback flows from your tools into ProductBridge. You can enable bidirectional sync to push status updates back when a linked roadmap item ships.
</Callout>
