## Capture Feedback from Support Channels

Your support team talks to users every day — and those conversations are full of product feedback. ProductBridge integrates with popular customer support platforms to automatically extract and import feedback from support tickets and conversations, so nothing gets lost.

## How Imported Feedback Works

All feedback imported from support tools is:

- **Tagged with its source** — You always know where feedback came from (e.g., "via Intercom", "via Zendesk")
- **Linked to a user** — If the support ticket has a user email, ProductBridge matches it to an existing user profile or creates a new one
- **Deduplicated** — If the same user submits similar feedback across multiple channels, ProductBridge groups them together
- **Available in Intelligence** — Imported feedback flows into [Feedback Intelligence](/core-concepts/feedback-intelligence) for automatic categorization and analysis

<Callout kind="info">
  Support integrations are one-way by default — feedback flows from support tools into ProductBridge. You can enable bidirectional sync to push status updates back to the support platform when a linked roadmap item ships.
</Callout>
