## Create Your Project

<Steps>
  <Step title="Go to Dashboard">
    Visit `https://dashboard.productbridge.io` and log in.
  </Step>
  <Step title="Start New Project">
    Click the "New Project" button at the top right.
  </Step>
  <Step title="Fill Project Details">
    Enter project name, description, and optionally invite team members.
  </Step>
  <Step title="Finish">
    Click "Create". Your project is now set up and ready to organize documentation.
  </Step>
</Steps>
