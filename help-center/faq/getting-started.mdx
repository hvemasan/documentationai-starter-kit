## Overview

To start using ProductBridge, create a project in your dashboard.

## Steps

1. Sign in at `https://dashboard.productbridge.io`
2. Click "New Project"
3. Name your project and add a description
4. Click "Create" to finish

Now your project is ready for documentation and team collaboration.
