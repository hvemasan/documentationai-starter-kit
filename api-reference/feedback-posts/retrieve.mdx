---
title: "Retrieve a feedback post"
description: "Look up the full details of a single feedback post by its id, scoped to the authenticated organization."
---

Retrieves the details of an existing feedback post, specified by its id. The post must belong to the authenticated organization — passing a UUID that exists in another org returns `404 not found`.

## Endpoint

```
POST https://api.productbridge.io/api/external/v1/feedback-posts/retrieve
```

## Arguments

<ParamField body="api_key" param-type="string" required="true" deprecated="false">
  Your organization's public API key. See [Authentication](/api-reference/authentication).
</ParamField>

<ParamField body="id" param-type="string" required="true" deprecated="false">
  The feedback post's UUID.
</ParamField>

## Returns

A [feedback post object](/api-reference/feedback-posts/overview#the-feedback-post-object) on success, or a 404 error envelope when the id doesn't exist for this organization.

## Example request

<CodeGroup show-lines="true" tabs="curl,JavaScript,Python">
  ```bash
  curl -X POST https://api.productbridge.io/api/external/v1/feedback-posts/retrieve \
    -H 'Content-Type: application/json' \
    -d '{
      "api_key": "pb_YOUR_PUBLIC_API_KEY",
      "id": "aa3c3ef8-b8cd-cd15-01ba-postpost0001"
    }'
  ```

  ```javascript
  const res = await fetch(
    "https://api.productbridge.io/api/external/v1/feedback-posts/retrieve",
    {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({
        api_key: process.env.PRODUCTBRIDGE_API_KEY,
        id: "aa3c3ef8-b8cd-cd15-01ba-postpost0001",
      }),
    }
  );
  if (res.status === 404) throw new Error("Feedback post not found");
  if (!res.ok) throw new Error(`HTTP ${res.status}`);
  const post = await res.json();
  ```

  ```python
  import os, httpx

  resp = httpx.post(
      "https://api.productbridge.io/api/external/v1/feedback-posts/retrieve",
      json={
          "api_key": os.environ["PRODUCTBRIDGE_API_KEY"],
          "id": "aa3c3ef8-b8cd-cd15-01ba-postpost0001",
      },
  )
  if resp.status_code == 404:
      raise LookupError("Feedback post not found")
  resp.raise_for_status()
  post = resp.json()
  ```
</CodeGroup>

## Errors

| Status | Body | Cause |
|---|---|---|
| `401` | `{"detail":{"error":"invalid api_key"}}` | Missing / unknown / inactive `api_key`. |
| `404` | `{"detail":{"error":"feedback post not found"}}` | Wrong id, deleted, or different organization. |
| `422` | Validation error envelope | Missing `id`, or `id` is not a UUID. |

See [Errors](/api-reference/errors) for the full envelope shape.
