---
title: "Retrieve a tag"
description: "Look up the full details of a single tag by its id, scoped to the authenticated organization."
---

Retrieves the details of an existing tag, specified by its id. The tag must belong to the authenticated organization — passing a UUID that exists in another org returns `404 not found`.

## Endpoint

```
POST https://api.productbridge.io/api/external/v1/tags/retrieve
```

## Arguments

<ParamField body="api_key" param-type="string" required="true" deprecated="false">
  Your organization's public API key. See [Authentication](/api-reference/authentication).
</ParamField>

<ParamField body="id" param-type="string" required="true" deprecated="false">
  The tag's UUID.
</ParamField>

## Returns

A [tag object](/api-reference/tags/overview#the-tag-object) on success, or a 404 error envelope when the id doesn't exist for this organization.

## Example request

<CodeGroup show-lines="true" tabs="curl,JavaScript,Python">
  ```bash
  curl -X POST https://api.productbridge.io/api/external/v1/tags/retrieve \
    -H 'Content-Type: application/json' \
    -d '{
      "api_key": "pb_YOUR_PUBLIC_API_KEY",
      "id": "773c3ef8-b8cd-cd15-01ba-tag00000001"
    }'
  ```

  ```javascript
  const res = await fetch(
    "https://api.productbridge.io/api/external/v1/tags/retrieve",
    {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({
        api_key: process.env.PRODUCTBRIDGE_API_KEY,
        id: "773c3ef8-b8cd-cd15-01ba-tag00000001",
      }),
    }
  );
  if (res.status === 404) throw new Error("Tag not found");
  if (!res.ok) throw new Error(`HTTP ${res.status}`);
  const tag = await res.json();
  ```

  ```python
  import os, httpx

  resp = httpx.post(
      "https://api.productbridge.io/api/external/v1/tags/retrieve",
      json={
          "api_key": os.environ["PRODUCTBRIDGE_API_KEY"],
          "id": "773c3ef8-b8cd-cd15-01ba-tag00000001",
      },
  )
  if resp.status_code == 404:
      raise LookupError("Tag not found")
  resp.raise_for_status()
  tag = resp.json()
  ```
</CodeGroup>

## Example response

```json
{
  "id": "773c3ef8-b8cd-cd15-01ba-tag00000001",
  "name": "mobile",
  "code": "mobile",
  "color": "#10B981",
  "show_on_feedback": true,
  "show_on_roadmap": true,
  "created_at": "2026-04-12T08:30:11.000Z"
}
```

## Errors

| Status | Body | Cause |
|---|---|---|
| `401` | `{"detail":{"error":"invalid api_key"}}` | Missing / unknown / inactive `api_key`. |
| `404` | `{"detail":{"error":"tag not found"}}` | Wrong id, deleted, or different organization. |
| `422` | Validation error envelope | Missing `id`, or `id` is not a UUID. |

See [Errors](/api-reference/errors) for the full envelope shape.
