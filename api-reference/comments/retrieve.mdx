---
title: "Retrieve a comment"
description: "Look up the full details of a single comment by its id, scoped to the authenticated organization."
---

Retrieves the details of an existing comment, specified by its id. The comment must belong to the authenticated organization.

## Endpoint

```
POST https://api.productbridge.io/api/external/v1/comments/retrieve
```

## Arguments

<ParamField body="api_key" param-type="string" required="true" deprecated="false">
  Your organization's public API key. See [Authentication](/api-reference/authentication).
</ParamField>

<ParamField body="id" param-type="string" required="true" deprecated="false">
  The comment's UUID.
</ParamField>

## Returns

A [comment object](/api-reference/comments/overview#the-comment-object) on success, or a 404 error envelope when the id doesn't exist for this organization.

## Example request

<CodeGroup show-lines="true" tabs="curl,JavaScript,Python">
  ```bash
  curl -X POST https://api.productbridge.io/api/external/v1/comments/retrieve \
    -H 'Content-Type: application/json' \
    -d '{
      "api_key": "pb_YOUR_PUBLIC_API_KEY",
      "id": "ee3c3ef8-b8cd-cd15-01ba-comment000001"
    }'
  ```

  ```javascript
  const res = await fetch(
    "https://api.productbridge.io/api/external/v1/comments/retrieve",
    {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({
        api_key: process.env.PRODUCTBRIDGE_API_KEY,
        id: "ee3c3ef8-b8cd-cd15-01ba-comment000001",
      }),
    }
  );
  if (res.status === 404) throw new Error("Comment not found");
  if (!res.ok) throw new Error(`HTTP ${res.status}`);
  const comment = await res.json();
  ```

  ```python
  import os, httpx

  resp = httpx.post(
      "https://api.productbridge.io/api/external/v1/comments/retrieve",
      json={
          "api_key": os.environ["PRODUCTBRIDGE_API_KEY"],
          "id": "ee3c3ef8-b8cd-cd15-01ba-comment000001",
      },
  )
  if resp.status_code == 404:
      raise LookupError("Comment not found")
  resp.raise_for_status()
  comment = resp.json()
  ```
</CodeGroup>

## Errors

| Status | Body | Cause |
|---|---|---|
| `401` | `{"detail":{"error":"invalid api_key"}}` | Missing / unknown / inactive `api_key`. |
| `404` | `{"detail":{"error":"comment not found"}}` | Wrong id, deleted, or different organization. |
| `422` | Validation error envelope | Missing `id`, or `id` is not a UUID. |

See [Errors](/api-reference/errors) for the full envelope shape.
