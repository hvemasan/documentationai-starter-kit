## Import Feedback from the Tools You Already Use

Your team and users communicate across many platforms — support tools, messaging apps, review sites, and project trackers. ProductBridge connects to these tools and automatically imports feedback so you do not have to copy and paste between systems.

Auto-ingestion runs continuously. New feedback from connected tools appears in your inbox within minutes, deduplicated and tagged with its source.

## How Auto-Ingestion Works

<Steps>
  <Step title="Connect a tool" icon="plug">
    Navigate to **Settings > Integrations** and connect the tools you want to import feedback from. Each integration has its own setup guide with authentication and configuration steps.
  </Step>
  <Step title="Configure import rules" icon="filter">
    Define which messages, tickets, or reviews get imported as feedback. Use tags, categories, channels, or AI-based detection to control the flow.
  </Step>
  <Step title="Feedback flows in automatically" icon="inbox">
    Once configured, feedback is imported continuously. Each item is tagged with its source (e.g., "via Intercom", "via G2") and linked to a user profile when possible.
  </Step>
</Steps>

## What Gets Imported

| Data | Description |
|------|-------------|
| Feedback content | The message, ticket body, or review text |
| Source tag | Which tool the feedback came from |
| User identity | Email, name, or profile linked when available |
| Metadata | Timestamps, star ratings, tags, or categories from the source |

<Callout kind="info">
  All imported feedback is deduplicated automatically. If the same user submits similar feedback across multiple channels, ProductBridge groups them together so you see consolidated demand, not noise.
</Callout>
