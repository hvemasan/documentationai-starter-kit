## Choose Your Embed Type

Pick the widget that fits how your product is built. Each type has a dedicated setup guide with copy-paste code for your framework.

<Columns cols="2">
  <Card title="Floating Button" href="/core-concepts/feedback-collection/widgets/floating-button" icon="message-square" horizontal={false}>
    A button fixed to the corner of every page. Clicks open a **popup** modal or a full-height **sidebar** panel. The most popular option — zero layout changes required.
  </Card>
  <Card title="Inline Embed" href="/core-concepts/feedback-collection/widgets/inline-embed" icon="app-window" horizontal={false}>
    Renders the widget directly inside a `<div>` on your page — no overlay, no floating button. Ideal for a dedicated `/feedback` page or a help center section.
  </Card>
  <Card title="Custom Trigger" href="/core-concepts/feedback-collection/widgets/custom-trigger" icon="mouse-pointer-click" horizontal={false}>
    Attach the widget to **any element** — your own nav link, help button, or keyboard shortcut. The default floating button is hidden; only your element opens the widget.
  </Card>
  <Card title="iFrame Embed" href="/core-concepts/feedback-collection/widgets/iframe-embed" icon="frame" horizontal={false}>
    A plain `<iframe>` tag — no SDK, no JavaScript. Works anywhere HTML embeds are supported: Notion, Webflow, Framer, CMSs, and static pages.
  </Card>
  <Card title="Direct URL" href="/core-concepts/feedback-collection/widgets/direct-url" icon="link" horizontal={false}>
    A shareable link to your feedback portal. No code at all. Drop it into emails, Slack messages, support tickets, or anywhere a link works.
  </Card>
  <Card title="Identity Verification" href="/core-concepts/feedback-collection/widgets/identity-verification" icon="shield-check" horizontal={false}>
    Automatically identify your logged-in users in any SDK embed type. Generate a signed JWT on your server — setup guides for Node.js, Python, PHP, Ruby, and Go included.
  </Card>
</Columns>

## Not Sure Which to Pick?

| I want to… | Use this |
|---|---|
| Add feedback to my app with minimal effort | [Floating Button](/core-concepts/feedback-collection/widgets/floating-button) |
| Show the widget inside my own page layout | [Inline Embed](/core-concepts/feedback-collection/widgets/inline-embed) |
| Use my own button or nav item to open it | [Custom Trigger](/core-concepts/feedback-collection/widgets/custom-trigger) |
| Embed in a CMS or no-code tool | [iFrame Embed](/core-concepts/feedback-collection/widgets/iframe-embed) |
| Share a link with no code | [Direct URL](/core-concepts/feedback-collection/widgets/direct-url) |
| Link feedback to logged-in user profiles | [Identity Verification](/core-concepts/feedback-collection/widgets/identity-verification) |
